A GUI to make TUIs

The default GUI makes a TUI that is used to launch this GUI and TUI.
(it basically runs `cad` for the client, and launches `deno` for the shell.)

GUI

Canvas and whatnot here!

End result:

This will output a code block to copy and paste in the terminal.

The block could consist of a stringified (TO|DQ)ML parameter to be pasted inline OR a file download where the path would be included.

Instructions will be provided as to how to connect each separate shell process to external file(s) the user wishes to use.

The code inside the logic directory will parse the block and run the files specified to complete the GUI to TUI process.
